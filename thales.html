THALES - APIS-HMI

Thales APIS - HMI

This was a gargantuan project that spanned over 2 years worth of work.

Being that it is confidential I am not able to really show what the project looks like per se. It basically is an UI for the users at the train stations to control the station's alerts, screens, lines and trains.

When I started on the project I was responsible for implementing the UI (mostly HMTL & CSS) according to a very rigid spec.

As time passed and the project kept getting bigger and bigger the communication between different design teams and developers it was decided that I would also have the responsibility of proposing new UI elements as well as make the necessary corrections to the spec and write new specifications for the new elements.

The most challenging part of it all was really getting everything pixel perfect and spot those tiny and hard to catch bugs that would appear when maintaining a code base as big as this one. It was also quite challenging to get to know how things need to be done in projects of this magnitude. All the procedures and bureaucracy were really eye opening.
