Easy Easy Apps App Maker

This is the app maker/editor for Easy Easy Apps which is a tool to make iOS and Android apps without code.

This was probably the biggest project I've ever worked on and it's still growing!

This is a project in which I was the lead product designer and was responsible for a number of things, namely:

  • Developing new App Templates and layouts for the core platform that could be used be 115K+ people.
  • Supported both back-office as well as mobile app front-end development.
  • Prototyping (Low-res and hi-def mockups)
  • UX Design (since creating the app to getting it on your phone)
  • UI Design (HTML5 & CSS3 using SASS)
  • Creation of app templates

It was very rewarding to work on this project because of its large dimension and reach.

We used mostly GWT to build everything and are now using some ReactJs for some screens also.

Given the GWT markup generated structure proved very challenging to style it but well worth it.
