Byclosure Website

This is the website for Byclosure, S.A. which is a SW development company specialized on custom-made web and cross-platform mobile apps.

This project consisted on a redesign of the older website which was pretty outdated at the time.

This is a project in which I was the lead designer and was responsible for a number of things, namely:

  • Prototyping (Low-res and hi-def mockups)
  • UX Design
  • UI Design (HTML5 & CSS3 using SASS)
  • Writing copy in english
  • SEO Optimization
  • Site speed optimization

To work on this project was something that at the time was quite challenging to implement because I was still getting to grip with my first CSS wizardry and things didn't come that easy. Additionally I also got to know the real bane of any front-end dev - Cross-browser issues!

All in all it was quite the learning experience.
