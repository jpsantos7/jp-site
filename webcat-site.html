Webcat Website & Automated Tests SW

Webcat is a tool for managing and automating tests and/or requirements which puts feature description at the center of all the phases of the software development cycle.

This is a project in which I was the lead product designer and was responsible for a number of things, namely:

  • Prototyping (Low-res and hi-def mockups)
  • UX Design (both site and SW tool)
  • UI Design (HTML5 & CSS3 using SASS)
  • Writing copy in english
  • SEO Optimization
  • Site speed optimization

This has been a very fast paced project that was done in big chunks of intense work and testing.

Below you can watch a video of the Webcat purr its way out of a series of specifications

For the website we used Ruby on Rails to get it up and running with Middleman.

For the tool itself I used HAML and LESS to implement the UI.

It was really challenging to get this project out as fast as it was but at the same time very rewarding because it is in fact a great tool.
