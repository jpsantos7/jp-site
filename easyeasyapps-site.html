Easy Easy Apps Website

This is the website for the Easy Easy Apps editor which is a SW tool to make iOS and Android apps without code.

It has a lot of integrations with many know providers such as Facebook, Instagram, LinkedIn, Wordpress.

This is a project in which I was the lead designer and was responsible for a number of things, namely:

  • Prototyping (Low-res and hi-def mockups)
  • UX Design (from the registry form to actually having an app installed on your phone)
  • UI Design (HTML5 & CSS3 using SASS)
  • Writing and translating copy in english, spanish and brazilian portuguese
  • SEO Optimization
  • Site speed optimization

It's been a blast working on this project and it's awesome to see it getting more and more results each day.

As of now the site managed to capture approximately 100K users who created apps and it's really good to see that users are able to get things done!
